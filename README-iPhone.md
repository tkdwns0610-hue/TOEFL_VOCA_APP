# TOEFL VOCA iPhone PWA

This folder contains the installable iPhone/PWA build.

## Install On iPhone

1. Upload this folder to an HTTPS host.
2. Open `index.html` in Safari.
3. Tap Share.
4. Tap Add to Home Screen.
5. Use the `TOEFL VOCA` icon from the home screen.
